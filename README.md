## Описание программы
Expense Tracker - это десктопное приложение с графическим интерфейсом для учета личных расходов. Программа позволяет добавлять, просматривать и фильтровать расходы, а также подсчитывать общую сумму трат за выбранный период.

### Основные возможности:
- Добавление расходов с указанием суммы, категории и даты
- Просмотр всех расходов в удобной таблице
- Фильтрация по категории и периоду
- Подсчет общей суммы расходов
- Сохранение данных в формате JSON
- Валидация вводимых данных
- Удаление записей
